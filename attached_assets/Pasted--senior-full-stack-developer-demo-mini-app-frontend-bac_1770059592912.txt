Экран 4 — КАК ЭТО РАБОТАЕТ  
(4 шага)

Экран 5 — TEXT DEMO  
ВАЖНО:
— Это демонстрация СЕРВИСА
— Пользователь не печатает
— Есть 3 кнопки:
  • Что это?
  • Кому подходит?
  • Где используется?
— По нажатию кнопки:
  → запрос уходит в backend
  → backend возвращает текст (через LLM или mock)
  → ответ появляется в чате

Экран 6 — LIVE FORMAT (объяснение)
— Объяснение, что есть live-формат
— Без запуска аватара

Экран 7 — LIVE СЦЕНАРИИ
— Кнопки:
  • Sales
  • Projects
  • Team
  • Expert
— При выборе:
  → создаётся sessionId
  → запрашивается контекст для live-аватара
  → показывается iframe / placeholder live-аватара

Экран 8 — ПИЛОТ
— Описание пилота
— Цена $2,500

Экран 9 — КОНТАКТ
— Форма (имя, контакт, сообщение)
— Отправка в backend
